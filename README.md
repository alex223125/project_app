### To run rubocop

```bundle exec rubocop```

### To run rubocop-rpsec

```rubocop --plugin rubocop-rspec```

### To run rubocop-rails

```rubocop --plugin rubocop-rails```

### To precompile tailwind css

```rake assets:precompile```
